Dynamically Typed Language

  • Types are associated with values not variables
  • The same variables can hold multiple values
  • We do not need to specify types
  • Most other languages are statically typed
  • There are subsets of JS and addons to allow static typing(TypeScript, Flow)

--------Primitive Types ---------

  1. String
  2. Number
  3. Boolean
  4. Null
  5. Undefined
  6. Symboles (ES6)

--------Reference Type ---------
  1. Arrays
  2. Objects literals
  3. Fucntions
  4. Dates
  5. Anything Else
